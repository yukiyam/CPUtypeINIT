# coding=Shift_JIS

CPUtypeINIT
===========

Mac OS の自称するCPU形式を PowerPC 603 に変更します。
AltiVec を強制的に無効化にしたい時とかに便利かと思います。
変更はシステム全体に影響します。（状況によって切り替える、みたいな器用なことはできません）

技術的には、 Gestalt() 関数の
  - gestaltNativeCPUtype
  - gestaltNativeCPUfamily
以上のセレクタに対する返却値を gestaltCPU603 に変更します。


必要システム条件
----------------

- 漢字Talk 7.5


使い方
------

システムフォルダの機能拡張フォルダに放り込んでください。


ビルド方法
----------

MPW と Pascal コンパイラが必要です。
CodeWarrior でも行けると思いますが、その場合はご自分で設定してください。

MPWに読み込ませる前に改行コードをCRにするといいかもしれません。
その場合は switchcrlf.pl を使うか、もしくは適当なテキストエディタを使ってください。

MPWを立ち上げたあとの手順は

1. カレントディレクトリをコードのあるフォルダに変更する
2. 新しい行に 'make' と入力し、Enter
3. 表示されたビルドコマンドを選択し、Enter

になります。


発展的な使い方
--------------

'CPUT' (128) リソースを編集することで Gestalt() の返却値をカスタマイズできます。
（ResEdit テンプレートを作ったので、開けば編集方法は分かると思います）
設定する値については Universal Headers の Gestalt.h を見てください。
（もしくは Gestalt.h でインターネット検索してください）

変更の反映には再起動が必要です。
